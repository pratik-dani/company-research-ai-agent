# Output Format

The Company Research & Analysis Agent provides output in two views: Overview (Company Report) and Raw Data.

## Overview View

The overview provides a formatted markdown report containing key insights about the company.

```json
{
  "domain": "string",
  "generated_report": "string (markdown)"
}
```

### Report Sections

The generated report includes:
1. Overview and Core Business
2. Products and Services
3. Market Presence and Performance
4. Key Personnel and Organization
5. Financial Metrics and Funding
6. Technology Stack and Digital Presence
7. Recent Developments and News
8. Summary and Key Insights

## Dataset Views

The actor provides two dataset views:

### 1. Overview View
```json
{
  "fields": ["domain", "generated_report"],
  "display": {
    "component": "table",
    "properties": {
      "domain": {
        "label": "Domain",
        "format": "string"
      },
      "generated_report": {
        "label": "Generated Report",
        "format": "string"
      }
    }
  }
}
```

### 2. Raw Data View
```json
{
  "fields": [
    "domain",
    "recent_news",
    "linkedin_data",
    "pitchbook_data",
    "crunchbase_data",
    "funding_analysis"
  ],
  "display": {
    "component": "table"
  }
}
```

<Note>
All dates in the output are in ISO 8601 format (YYYY-MM-DD).
Monetary values are in USD unless otherwise specified.
</Note>
